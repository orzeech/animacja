Info

This is just a demo page to show how a guitar can be customized online in a 3D environment. Only a few options are working. If you want to see the potential for this webapp, please visit Norita Guitars current 2D guitar design webapp.

If you are a guitar manufacturer or you want to sell guitars online, this webapp can be integrated into an e-commerce site, here is an example: Halo Guitars Customization Tool. Feel free to ask for a budget if you are interested in a custom development for your site, please contact jorge@jipijopo.com.

Compatibility:

  • WebGL 1.0 is supported in the stable releases of most major browsers on both desktop and mobile platforms. Chrome, Firefox, Internet Explorer, Opera, and Safari are all known to have good WebGL support on both desktop and mobile browsers. See http://caniuse.com/#feat=webgl for availability details.
  • Technical issues such as known hardware problems or lack of required GPU features may prevent WebGL from running in some cases. (www.khronos.org)
